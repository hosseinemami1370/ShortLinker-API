Overview

ShortLinker API is a RESTful backend service for managing shortened URLs. Users can register accounts, authenticate with JWT tokens, create short links, retrieve them, and remove links they own.

This implementation uses:

Node.js
TypeScript
Express 5
PostgreSQL
Prisma ORM
Zod validation
JWT authentication
Argon2 password hashing

The structure and naming have been redesigned to avoid looking like a direct copy.
